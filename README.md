# Plumbing Services Website

A React + TypeScript landing website for plumbing services.

## Live Demo

https://asushko25.github.io/app-servis/

## Features

- Responsive layout (mobile-friendly)
- Services section with clear presentation
- Modern UI components and clean styling
- Fast build and deployment with Vite
- SCSS structure for scalable styling

## Tech Stack

- React
- TypeScript
- SCSS
- Vite

## Getting Started

Clone the repository:

git clone https://github.com/asushko25/app-servis.git
cd app-servis

Install dependencies:

npm install

Run locally:

npm run dev
